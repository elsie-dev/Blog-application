## User Flask API & SQL Alchemy


## Quick Start 

``` bash
# Create & Activate venv
$ python3 -m venv venv
$ . venv/bin/activate
```
## Install dependencies
```bash
$ pip3 install -r requirments.txt
```
## Create DB
```python
$ python3
>> from app import db
>> exit()
```
## Run Server (http://127.0.0.1:5000/)
```bash

python app.py
```
